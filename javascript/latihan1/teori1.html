Apa Itu Pemograman?

Sebelum Kepemograman maka kita harus mengetahui apa itu program

Program adalah Kumpulan atau serangkaian instruksi dengan tujuan untuk menyelesaikan sebuah pekerjaan
Apakah Aplikasi Termasuk Program?
Iya Termasuk Program Karena didalam Aplikasi terdapat suatu kumpulan instruksi

Instruksi dalam Pemograman
  1. Memberikan Perintah Pada Komputer
  2. Melalui Bahasa Pemmograman
  3. Memecahkan tugas instruksi menjadi sederhana

Instuksi Sederhana ?

  • Melakukan Operasi matematika
  • menampilkan tulisan pada layar
  • Mengubah Ukuran, warna, posisi dari sebuah pixel

    • Perilaku komputer

    1. Komputer Tidak Begitu
    2. Penurut
    3. tidak kenal lelah

    Statemen ?

    Statemen adalah perintah untuk komputer dalam bahasa pemograman

    Contoh Bahasa Pemograman

  • Pascal umur :=30
